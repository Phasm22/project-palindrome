## Summary

- **No critical missing:** Everything the app *requires* is either set in your `.env` or has a default.
- **Worth adding if you use those features:**
  - **Terraform on yin/yang:** `PROXMOX_YIN_TF_SECRET`, `PROXMOX_YANG_TF_SECRET` (or keep using `PROXMOX_CLUSTER_TF_SECRET` if one secret is used for both).
  - **Create VM / Terraform key injection:** `SSH_PUBLIC_KEY=...` (env-validator warns if unset).
  - **LLM intent classifier model:** `INTENT_CLASSIFIER_MODEL=gpt-4o-mini` (or another model) if you want to override the default.
- **Optional:** All other gaps have in-code defaults or only affect scripts/optional tools.

Your `.env` has **OPENAI_MODEL**; the codebase does not read `process.env.OPENAI_MODEL` anywhere (runner uses a constant model id). So that key is unused, not a gap.
