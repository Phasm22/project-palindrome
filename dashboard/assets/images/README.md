# Logo Assets

## Place your logo here

1. **Main Logo**: Place your 1kx1k PNG logo file here and name it `logo.png`
   - File should be: `dashboard/assets/images/logo.png`
   - Recommended: 1000x1000px PNG with transparency
   - The code will automatically check and fix transparency if needed

## File Structure
```
dashboard/assets/images/
  ├── logo.png          (your main logo - 1kx1k PNG)
  └── README.md         (this file)
```

## Transparency Check

The system will automatically:
- Check if the PNG has proper transparency
- Fix transparency using ImageMagick if needed
- Use the logo throughout the site with animations
